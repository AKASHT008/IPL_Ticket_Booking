# 🏏 IPL Ticket Booking System

A full-stack web application developed using Python Flask and MySQL for online IPL ticket booking.

## 🚀 Features

- User Registration & Login
- IPL Match Listing
- Online Ticket Booking
- Razorpay Payment Integration
- QR-Code-Based Digital Ticket
- User Dashboard
- Responsive Bootstrap UI

## ⚙️ How to Run the Project

### 1 Clone Repository

```bash
git clone https://github.com/AKASHT008/IPL_Ticket_Booking.git
```

### 2 Install Requirements

```bash
pip install -r requirements.txt
```

### 3 Configure Database

- Open MySQL Workbench
- Create database
- Import `ipl_booking.sql`

### 4 Run Flask Application

```bash
python app.py
```
